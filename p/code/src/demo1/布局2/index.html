销售的第一大法则?

张宁   2016年10月21日14:39

“销售产品时不要给别人说不的机会。”

销售的第一大法则是，如果一个人空手而归，他们很可能不会回来了。这就是为什么你必须要积极地对待客户，但你必须确保分寸以免把他们吓跑。

你能保持谈话的一个方法是避免问答案为有或没有的问题。例如，如果你推销一个服务计划，不要问“你对我们3或5年的服务计划感兴趣吗？”相反，问“你对3年服务计划或是5年计划是否感兴趣，哪个是更好的价值？”乍一看，他们似乎是同一个问题，当一个客户能否摆脱第一个问题，却很难拒绝第二个问题， 因为他们不能仅仅回答“不.”
